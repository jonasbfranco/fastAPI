# Criando API

### Criar ambiente virtual
```shell
python -m venv venv  
```

### Ativando ambiente virtuall no Windows
```shell
venv/Scripts/activate 
```

### Instalando fastAPI
```shell
pip install "fastapi[standard]"
```

### Atualizar pip
```shell
python.exe -m pip install --upgrade pip
```

### Executar fastAPI pip
```shell
fastapi dev main.py
```
